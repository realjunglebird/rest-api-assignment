# Простой REST API для работы с задачами

## Описание проекта
Реализован REST API для работы с задачами. Программа представлена в виде локального HTTP-сервера, 
функционирующего на порте 8080. 

Поддерживаются следующие функции:
- Создание новой задачи
- Получение списка всех задач
- Получение одной задачи по идентификатору
- Получение списка задач по статусу (активные/завершенные)
- Обновление задачи по идентификатору
- Удаление задачи по идентификатору

Модель задачи содержит следующие поля:
- Уникальный идентификатор [Long]
- Название [String]
- Описание [String]
- Состояние (активна/завершена) [boolean]
- Время создания задачи [LocalDateTime]
- Время обновления задачи [LocalDateTime]

## Технические подробности реализации
Задание выполнено на языке Java (Java 21) с применением фреймворка Spring Boot 4.
Работоспособность получившегося API проверялась с помощью Postman.  
Реализовано 4 класса:
- **Task** - модель данных задачи
- **TaskRequest** - DTO для отделения API модели от модели данных
- **TaskService** - сервис, работающий с хранилищем данных
- **TaskController** - контроллер, обрабатывающий HTTP-запросы

## Примеры использования API
### Создание новой задачи:
- Метод: POST  
- URL: http://localhost:8080/tasks  
- Тело запроса:
  - Тип: JSON
  - Содержимое:
    ```json
      {
        "title": "Купить продукты",
        "description": "Молоко, хлеб, масло, яйца",
        "completed": false
      }
      ```

### Получение списка всех задач:
- Метод: GET
- URL: http://localhost:8080/tasks

### Получение одной задачи по идентификатору
- Метод: GET
- URL: http://localhost:8080/tasks/1

### Получение списка задач по статусу (активные/завершенные)
- Метод: GET
- URL: http://localhost:8080/tasks/status/true

### Обновление задачи по идентификатору
- Метод: PUT
- URL: http://localhost:8080/tasks/1
- Тело запроса:
    - Тип: JSON
    - Содержимое:
      ```json
        {
          "title": "Купить вкусные продукты",
          "description": "Шоколад, печенье, мармелад",
          "completed": true
        }
        ```

### Удаление задачи по идентификатору
- Метод: DELETE
- URL: http://localhost:8080/tasks/1
